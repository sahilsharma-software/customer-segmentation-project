# Customer Segmentation using K-Means Clustering

## Project Overview

This project performs customer segmentation for a mall dataset using K-Means Clustering.
Customers are grouped based on their Annual Income and Spending Score to identify different customer segments.

## Objective

The goal of this project is to identify different groups of customers so that businesses can understand their spending behaviour and design better marketing strategies.

## Dataset

The dataset contains the following features:

* CustomerID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1–100)

## Steps Performed

1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Outlier Detection and Removal
5. Exploratory Data Analysis (EDA)
6. Feature Scaling
7. K-Means Clustering
8. Finding Optimal Clusters using Elbow Method
9. Cluster Evaluation using Silhouette Score
10. Cluster Visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Algorithm Used

K-Means Clustering (Unsupervised Machine Learning)

## Evaluation Method

Silhouette Score was used to evaluate the clustering performance.

## Results

Customers were successfully segmented into different clusters based on income and spending patterns.

Example customer segments include:

* High Income – High Spending (VIP Customers)
* High Income – Low Spending (Target Customers)
* Low Income – High Spending
* Low Income – Low Spending
* Average Customers

## Visualization

Cluster visualization was performed using scatter plots to clearly show different customer groups.

## Conclusion

Customer segmentation helps businesses understand customer behaviour and improve marketing strategies by targeting specific groups of customers.
